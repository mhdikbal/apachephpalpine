# Apache2 + PHP + Alpine
# How to use
</p>
1. Pull image to local machine

``
docker pull parewa/apache2phpalpine:v1
``

</p>
2. then run 
</p>

``
docker run  -dp 80:80 --name apachephpalpine parewa/apache2phpalpine:v1
``
</p>
